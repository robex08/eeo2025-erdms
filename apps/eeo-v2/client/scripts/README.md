# Scripts Directory

Obsahuje utility skripty a debug nástroje pro projekt.

## Struktura

### debug/
- Debug skripty pro testování a diagnostiku
- F5 persistence testy
- Storage audity a šifrování testy
- Draft detection nástroje

### cleanup/
- Cleanup skripty pro odstranění debug logů
- Corrupted drafts cleanup
- Git commit utilities
- Tooltips search nástroje

## Použití

Tyto skripty jsou pomocné nástroje pro vývoj a údržbu aplikace.
Většina z nich byla použita během vývoje pro debugging a cleanup operace.
